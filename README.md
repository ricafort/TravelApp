# TravelApp

An offline-first travel application built with Flutter.

## Features

- **Offline-First Mechanics**: Robust travel utilities that function without internet access.
- **Voice Journaling**: Record experiences on the go.
- **Localized Survival Phrases**: Essential phrases for smooth communication.
- **Adaptive Packing Lists**: Context-aware recommendations for packing.
- **Currency Conversion**: Context-aware and offline-capable currency conversion.

## Getting Started

1. Ensure you have Flutter installed.
2. Clone this repository.
3. Run `flutter pub get` in the `travelapp` directory to install dependencies.
4. Run the app using `flutter run` or via your IDE.
